# dotfiles

Personal dotfiles including:

1. git
2. vim
3. starship
4. fish
5. wezterm

I code with neovim, also check out my [neovim configuration](https://github.com/yilinfang/nvim) and [bootstrap scripts](https://github.com/yilinfang/nvim-starter).

## Usage

```bash
git clone https://github.com/yilinfang/dotfiles
cd dotfiles
chmod +x intall.sh
./install.sh
```
