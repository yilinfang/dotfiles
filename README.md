# dotfiles

Personal dotfiles including:

- Alacritty (deprecated)
- Fish
- Ghostty
- Git
- Starship
- Vim
- WezTerm (deprecated)
- Zsh (deprecated)

I code with an Neovim-based PDE([Personalized Development Environment](https://youtu.be/QMVIJhC9Veg?si=VgJQLBVTIYmNjVSD)), also check my [Neovim configuration](https://github.com/yilinfang/nvim-ks) and [PDE bootstrap scripts](https://github.com/yilinfang/pde-starter) for more details.

## Usage

```bash
git clone https://github.com/yilinfang/dotfiles.git
cd dotfiles
chmod +x intall.sh
./install.sh
```
