# Dotfiles

Personal dotfiles including configurations for:

- Alacrittly (deprecated)
- Fish
- Ghostty (deprecated)
- Git
- Starship
- Tmux
- Vim
- WezTerm
- Yazi
- Zellij (deprecated)
- Zsh (deprecated)

## Usage

```bash
git clone https://github.com/yilinfang/dotfiles.git
cd dotfiles
git fetch --all
git checkout vsc
chmod +x install.sh
./install.sh
```
