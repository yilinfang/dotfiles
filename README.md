# Dotfiles

Personal dotfiles including configurations for:

- Alacritty
- Fish
- Ghostty
- Git
- Neovim
- Starship
- Vim
- WezTerm (deprecated)
- Yazi
- Zellij
- Zsh (deprecated)

I code with _VIM_ and _VSCode_. The Neovim configuration is primarily for the _VSCode Neovim_ plugin.

## Usage

```bash
git clone https://github.com/yilinfang/dotfiles.git
cd dotfiles
git fetch --all
git checkout vsc
chmod +x install.sh
./install.sh
```
