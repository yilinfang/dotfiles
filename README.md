# dotfiles

Personal dotfiles including:

- git
- vim
- neovim
- zellij
- fish
- starship
- kitty
- wezterm

I code with vim and VSCode, the Neovim configuration is mainly for VSCode-neovim plugin.

## Usage

```bash
git clone https://github.com/yilinfang/dotfiles
cd dotfiles
chmod +x intall.sh
./install.sh
```
