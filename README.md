# Dotfiles

Personal dotfiles including configurations for:

- Fish
- Ghostty
- Git
- Kitty
- Neovim
- Starship
- Vim
- WezTerm
- Zellij

I code with VIM and VSCode. The Neovim configuration is primarily for the VSCode-neovim plugin.

## Usage

```bash
git clone https://github.com/yilinfang/dotfiles.git
cd dotfiles
git fetch --all
git checkout vsc
chmod +x install.sh
./install.sh
```
