# Dotfiles

Personal dotfiles including configurations for:

- Alacrittly (deprecated)
- Fish
- Ghostty
- Git
- Neovim
- Starship
- Tmux (deprecated)
- Vim
- WezTerm (deprecated)
- Yazi (deprecated)
- Zellij
- Zsh (deprecated)

I code with Vim and VSCode, the Neovim configuration is mainly for VSCode-Neovim extension.

## Usage

```bash
git clone https://github.com/yilinfang/dotfiles.git
cd dotfiles
git fetch --all
git checkout vsc
chmod +x install.sh
./install.sh
```
