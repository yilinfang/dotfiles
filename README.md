# dotfiles

Personal dotfiles including:

- git
- vim
- neovim
- zellij
- fish
- starship
- Ghostty
- kitty
- wezterm

I code with vim and VSCode, the Neovim configuration is mainly for VSCode-neovim plugin.

## Usage

```bash
git clone https://github.com/yilinfang/dotfiles.git
cd dotfiles
git fetch --all
git checkout vsc
chmod +x intall.sh
./install.sh
```
